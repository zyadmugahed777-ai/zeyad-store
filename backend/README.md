# زياد للأعمال - واجهة الواجهة الخلفية (Backend)

هذا المشروع يمثل الواجهة الخلفية (Backend) لمتجر "زياد للأعمال" ومبني وفق معمارية "Guest First Architecture" التي تتيح للعملاء استخدام كافة مميزات المتجر دون الحاجة لإنشاء حساب مسبق.

## التقنيات المستخدمة

- **Node.js & Express.js**: إطار العمل الأساسي.
- **SQLite (better-sqlite3)**: قاعدة بيانات خفيفة وسريعة، تعمل بنمط متزامن ومجهزة بنمط WAL للأداء العالي.
- **EJS**: محرك قوالب لبناء واجهات لوحة تحكم الإدارة (Admin Dashboard).
- **Multer**: لرفع وإدارة الملفات.
- **Bcrypt.js**: لتشفير كلمات المرور الخاصة بمدراء النظام.

## التثبيت والتشغيل

1. تثبيت الاعتماديات:
   \`\`\`bash
   npm install
   \`\`\`

2. تشغيل السيرفر (التطوير):
   \`\`\`bash
   npm run dev
   \`\`\`
   سيقوم السيرفر تلقائياً بإنشاء قاعدة البيانات وإدراج البيانات الأولية إذا لم تكن موجودة.

3. تسجيل الدخول للوحة الإدارة:
   - الرابط: \`http://localhost:3000/admin/login\`
   - اسم المستخدم الافتراضي: \`admin\`
   - كلمة المرور الافتراضية: \`changeme123\`

## الخصائص الأساسية

1. **Guest First Architecture**: جميع عمليات الشراء والطلبات لا تتطلب تسجيل دخول. يتم إنشاء سجلات العملاء وربطها برقم الهاتف تلقائياً عند الطلب.
2. **الربط مع الواتساب**: كل طلب يقوم بإنشاء رابط رسالة واتساب منسقة، جاهزة للإرسال للعميل، وتظهر في تفاصيل الطلب في لوحة الإدارة.
3. **أداء عالي**: استخدام \`better-sqlite3\` مع تفعيل خيارات الأداء التلقائية (WAL, Foreign Keys) يضمن سرعة تنفيذ عالية جداً.
4. **لوحة تحكم مدمجة**: لوحة تحكم سريعة وعملية بدون تعقيدات، مبنية بالـ Server-Side Rendering باستخدام EJS لتكون سريعة جداً.
